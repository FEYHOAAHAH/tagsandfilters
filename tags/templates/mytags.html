Django Template Примеры тегов

{ load }

{ load } Тег используется для загрузки тегов и фильтров шаблона Syntax: {load some_library }

{ templatetag }

{ templatetag } тег используется для вывода символов, используемых для разделения тегов и переменных Syntax: { templatetag openvariable }, { templatetag closevariable }

{ include }

The { include } тег используется для включения содержимого другого шаблона Syntax: { include 'another_template.html' }

{ extends }

The { extends } Тег используется для расширения базового шаблона Syntax: { extends 'base_template.html' }

{ comment }

The { comment } тег используется для добавления комментариев в шаблонах Syntax: { comment "This is a comment" } ... { endcomment }

{ resetcycle }

The { resetcycle } tag is used to reset the cycle tag state. Syntax: { resetcycle variable_name }

{ widthratio }

The { widthratio } тег используется для сброса состояния тега цикла Syntax: { widthratio value max_value max_width }

{ regroup }

The { regroup } Тег используется для группировки списка похожих объектов по общему атрибуту. Syntax: { regroup list by attribute as grouped_list }

Django Template примеры фильтров

{{ some_variable|date:"D d M Y" }}

The |date Фильтр используется для форматирования даты

{{ some_list|slice:":2" }}

The |slice Фильтр используется для разрезания списка

{{ some_text|lower }}

The |lower фильтр используется для преобразования текста в нижний регистр

{{ some_text|cut:"substring" }}

The |cut Фильтр используется для удаления указанной подстроки из строки

{{ some_text|linebreaksbr }}

The |linebreaksbr фильтр используется для преобразования разрывов строк в разрывы строк HTML

{{ some_text|length_is:"10" }}

The |length_is Фильтр используется для проверки того, равна ли длина строки указанному значению

{{ some_text|upper }}

The |upper фильтр используется для преобразования текста в верхний регистр

{{ some_variable|add:5 }}

The |add Фильтр используется для добавления значения к переменной

{{ some_text|rjust:"10" }}

The |rjust фильтр используется для выравнивания строки по правому краю в поле заданной ширины

{{ some_time|time:"H:i" }}

The |time фильтр используется для форматирования времени

{{ some_html|safe }}

The |safe фильтр используется для пометки строки как безопасной для вывода HTML

{{ some_float|floatformat:"2" }}

The |floatformat фильтр используется для форматирования числа с float числом

{{ some_text|wordcount }}

The |wordcount фильтр используется для подсчета количества слов в строке

{{ some_text|slugify }}

The |slugify Фильтр используется для преобразования строки в URL-адрес

{{ some_text|urlize }}

The |urlize фильтр используется для преобразования URL-адресов в тексте в кликабельные ссылки

{{ some_text|truncatechars:"50" }}

The |truncatechars фильтр используется для усечения строки до указанного количества символов
